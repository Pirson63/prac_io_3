# Task 3

## Необходимые пояснения по решению задачи приведены в файле ipynb.
## Ниже кратко приведены описания всех функций, использованных в решении.

# get_sold_items(sell)
Возвращает число проданных в этот день яблок и карандашей.
Редактирует датафрейм для удобства дальнейшей работы с ним.

# daily_inventory(supply, sell)
Определяет состояние склада на каждый день.
Суммирует все произошедшие поставки и вычитает
общее количество проданных товаров к
текущему дню.

# monthly_theft(daily_inv, inventory)
Определяет суммарное количество товара,
украденного в течение данного месяца.
Разность теоретического и фактического состояния склада
даёт понимание о количестве сворованного товара
до конца текущего месяца.
Чтобы найти количество украденного инвентаря в данный месяц,
нужно вычесть аналогичное значение для предыдущего месяца.

# update(state, sell, steal, dframe)
Формирует агрегированные данные об объемах продаж
и количестве сворованной продукции по штату и году.

# main(name, dframe)
Основная функция в программе. Собирает воедино данные из
всех остальных функций.
